# Physics-Exercises
In this repository I will put all physics exercises written in LaTex. 
There will be errors which, hopefully, I will be able to correct as soon as possible. Furthermore, as it happens so much, there will be exercises which I might have solved, but I don't like the solution or I think one can have edge cases etc.
Therefore, with this in mind, I hope that these exercises and ideas, which will be generated while solving them, will inspire many discussion between you (reader) and me.
